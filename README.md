# 🎓 Student Management System

## 📌 Description
This is a simple Student Management System built using Python and Object-Oriented Programming (OOP) concepts.  
It allows users to add student details and display them.

## ✨ Features
- Add multiple students
- Store student name and marks
- Display all student details
- Uses classes and objects (OOP)

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Open the project in PyCharm or any IDE
2. Run the Python file
3. Enter number of students
4. Enter details when asked
5. View student data output
